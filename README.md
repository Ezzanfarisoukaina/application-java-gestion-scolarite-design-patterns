# 🎓 Gestion Scolarité

## 📌 Description
Application web Java basée sur l’architecture MVC permettant l’authentification des utilisateurs et la gestion des rôles (admin / étudiant). Déployée sur Apache Tomcat avec une base de données MySQL.

## 🛠️ Technologies
- Java (Servlets / JSP)
- MySQL
- HTML / CSS
- Apache Tomcat
- JDBC

## ⚙️ Fonctionnalités
- Login utilisateur
- Gestion des sessions
- Redirection selon rôle
- Séparation MVC (Model / View / Controller)


   ## 🚀 How to Run the Project

### 1. Prérequis
- Java JDK 8+
- Eclipse IDE (ou IntelliJ)
- Apache Tomcat 9
- MySQL Server

---

### 2. Importer le projet
- Ouvrir Eclipse
- File → Import → Existing Dynamic Web Project
- Sélectionner le projet

---

### 3. Configurer le serveur
- Ajouter Apache Tomcat 9 dans Eclipse
- Associer le projet au serveur

---

### 4. Base de données
- Créer une base MySQL
- Importer le fichier SQL (scolarite.sql)
- Vérifier les identifiants dans `DBConnection.java`

---

### 5. Lancer le projet
- Click droit sur le projet
- Run on Server → Tomcat

---
